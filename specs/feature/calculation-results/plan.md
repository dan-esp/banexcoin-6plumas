# Implementation Plan — Calculation Results

## Objective

Build the review surface for grouped cashback outputs after calculation completes.

## Recommended Build Order

### 1. Totals Summary

- add summary cards or panels for batch totals

### 2. Results Table

- add account results table with core financial columns

### 3. Trace Detail

- add drawer or side panel for row-level or grouped calculation trace details

## Done Criteria

- a reviewer can inspect total liability and account-level outputs from one workflow step
- the screen leads naturally into finance approval rather than export
